# Proiect Qwirkle - Instrucțiuni de Rulare

## 1. Cerințe de Sistem (Librării)
Pentru a rula acest proiect, este necesar să aveți instalate librăriile **NumPy = 2.3.5** și **OpenCV = 4.12.0**.

Comandă de instalare:
```bash
pip install numpy opencv-python
```
## 2. Configurare Script

Singurele modificări necesare pentru a rula codul pe o altă mașină sunt la liniile 5 și 6 din scriptul principal:

Linia 5 (data_file_path): Modificați variabila cu calea (path-ul) către folderul care conține imaginile de test (input).

Linia 6 (output_dir): Modificați variabila cu calea (path-ul) către folderul unde doriți să fie salvate fișierele text cu rezultatele (output).
